# ポップアップレストラン「YUKASHI」サイト

6日間限定で運営されたポップアップレストラン「YUKASHI」の公式ホームページです。  
現在サイトは閉鎖されていますが、制作成果は GitHub Pages 上にアーカイブしています。

🔗 https://nanami-matsui.github.io/yukashi.github.io/

---

## 概要
本サイトは、「The Blue Camp」の活動の一環として運営された  
学生のみで運営する期間限定ポップアップレストラン「YUKASHI」の情報発信を目的として制作しました。

レストランに込めた想いや活動内容を、より多くの人に分かりやすく伝えるための
ホームページとなっています。

---

## 制作背景
チーム内には料理人志望者や水産に詳しいメンバーが多く在籍していて  
このレストラン運営にあたり、水産や食に関する専門知識を持たない私は、自分にできる形での貢献方法を模索しました。

その中で、**ホームページを通して活動や想いを発信すること**が  
チーム全体への貢献につながると考え、ホームページ作成を担当しました。

---

## 使用技術
- HTML  
- CSS  
- JavaScript  
- slick（画像スライダー）

---

## 具体的な取り組み・工夫点
### メンバー紹介のポップアップ表示
メンバー紹介部分をクリックすると詳細がポップアップ表示されるように実装し、  
来訪者にチームの想いや背景をより深く知ってもらえるよう工夫しました。

### レスポンシブデザイン
スマートフォン・タブレット・PCなど、  
異なる画面サイズでも快適に閲覧できるよう、  
スクリーンサイズに応じて画像やレイアウトが変化する設計にしました。

### 画像のスライド表示
`slick` を使用し、研修や活動の様子をスライドショー形式で表示することで、  
視覚的にも楽しめるサイトを目指しました。

### サーバー・ドメイン設定
運営期間やコストを考慮し、レンタルサーバー会社の選定から  
ドメイン取得までを担当しました。  
グループメンバーの承認を得たうえで決定し、  
サイト内には予約ページへのリンクも設置しました。

### セキュリティ対策
予約ページへのリンクを掲載するにあたり、  
SSLを設定し、セキュリティ面にも配慮しました。

## この制作を通して得たこと
初めてのホームページ制作であったため、  
HTML / CSS の知識に加え、  
**デザインの考え方やサーバーの仕組みについても自学で学習**しました。

技術面だけでなく、  
「チームの中で自分がどう貢献できるか」を考え、  
形にする経験を得ることができました。
